Kal'tsit State Machine 仪表盘 使用指南 状态历史 事件历史 关键记录 设定管理 人格演化 向量管理

这是什么

凯尔希状态机是一个面向对话场景的记忆与状态系统，支持状态推演、对话反思、记忆召回、阶段回顾和自动化整理。

主要功能

仪表盘
查看最新快照、环境信息、自动化执行报告与 Token 汇总。
状态历史
浏览与检索快照，支持批量删除和导出。
事件历史
管理事件锚点，支持编辑、分类筛选、归档视图和批量操作。
关键记录
保存可复用的长期事实（日期、事项、协作、医疗建议）。
设定管理
维护 L1/L2 设定、Prompt 模板、运行参数和导入能力。
人格演化/向量管理
执行演化预览与应用，管理向量同步、压缩和重建。

快速开始

1. 安装依赖：`pip install -r requirements.txt` 2. 配置模型：在 `config.yaml` 填写 LLM 参数 3. 启动服务：`python -m server.main` 4. 打开页面：`http://localhost:8000`
